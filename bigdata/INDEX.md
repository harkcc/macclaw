# Big Data Drive Index

Confirmed standard locations:
- Volume: `OpenClawData`
- Big objects root: `/Volumes/OpenClawData/bigdata/`

Source of truth: `INDEX.csv`.

## How to add an entry

Add a row to `INDEX.csv` with:
- name
- type (dataset|db|assets|backup)
- volume (Finder volume name)
- path (path on the external drive)
- project
- owner
- notes
- created_at / updated_at
